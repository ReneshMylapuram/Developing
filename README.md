# Advanced Equities Website

Multi-page website for Advanced Equities (AE), a student-run investment and research organization.

## Files

- `index.html` - home page
- `leadership.html` - leadership page
- `portfolio-tracker.html` - portfolio tracking page
- `portfolio-intelligence.html` - portfolio intelligence page
- `sector-research.html` - sector research page
- `internship-apply.html` - internship and applications page
- `events-updates.html` - events and updates page
- `styles.css` - visual design, responsive layout, and animations
- `script.js` - scroll reveal interactions

## Local preview

Open `index.html` in a browser.
